# Accomplishments — Personal Website Frontend

## Typed content-driven portfolio SPA

**Technologies:** Vue 3, TypeScript, Vite

**Role:** Personal portfolio SPA where homepage sections and project case studies are driven by typed content modules and composed section components.

**Advantages:** Editorial updates stay in TypeScript content files with compile-time validation instead of a CMS for a single-author site.

**Resume statement:** Used Vue 3, TypeScript, Vite as a personal portfolio SPA where homepage sections and project case studies are driven by typed content modules and composed section components, leveraging editorial updates in TypeScript content files with compile-time validation instead of a CMS for a single-author site.

## GitHub Pages static deployment

**Technologies:** GitHub Actions, Vite

**Role:** Automated build-and-deploy pipeline producing static `dist/` output to GitHub Pages with custom domain CNAME.

**Advantages:** Zero server maintenance; push-to-master triggers type-checked production build and Pages deploy.

**Resume statement:** Used GitHub Actions, Vite as an automated build-and-deploy pipeline producing static dist output to GitHub Pages with custom domain CNAME, leveraging zero server maintenance with push-to-master type-checked builds.
